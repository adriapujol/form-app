# Wedding Invitation form app

This app gets the information from the guests via form. Frontend React/SaSS and backend with Node/Express.

You can see it [here](https://jennifer-et-carlos.netlify.app/)

## Features

* Login system via JWT Tokens.
* Form to get users input with option to have a Plus One and up to 4 kids.
* Multiple Languages (FR/ES)
* Admin page to manage users. You can add, delete, order users by name and download the data in Excel format.
